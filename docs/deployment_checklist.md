# CNEC 한국 사업자등록정보 인증 시스템 배포 체크리스트

## 배포 전 확인사항

### 환경 변수 설정
- [ ] `.env` 파일에 실제 Supabase URL 및 Anon Key 설정
- [ ] Netlify 환경 변수에 `VITE_SUPABASE_URL` 설정
- [ ] Netlify 환경 변수에 `VITE_SUPABASE_ANON_KEY` 설정
- [ ] Netlify 환경 변수에 `BUSINESS_API_KEY` 설정 (사업자등록번호 API 키)

### Supabase 설정
- [ ] `supabase/schema.sql` 파일의 SQL 스크립트를 Supabase SQL Editor에서 실행
- [ ] RLS(Row Level Security) 정책 적용 확인
- [ ] 초기 관리자 계정 생성 (스크립트 또는 수동으로)

## 배포 단계

### 빌드 및 배포
1. 로컬에서 빌드 테스트
   ```bash
   npm run build
   ```

2. Netlify 배포 설정
   - GitHub 저장소 연결
   - 빌드 명령어: `npm install && npm run build`
   - 배포 디렉토리: `dist`
   - 환경 변수 설정

3. 배포 후 확인
   - 배포된 사이트 접속 테스트
   - 로그인 및 주요 기능 테스트
   - 콘솔 오류 확인
