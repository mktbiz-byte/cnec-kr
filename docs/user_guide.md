# CNEC 한국 사업자등록정보 인증 시스템 사용자 가이드

## 1. 시작하기

CNEC 한국 사업자등록정보 인증 시스템에 오신 것을 환영합니다. 본 시스템은 사업자등록번호를 기반으로 간편하게 회원가입 및 로그인을 하고, 기업 정보를 관리할 수 있는 서비스를 제공합니다.

## 2. 회원가입

1. 메인 페이지에서 **[회원가입]** 버튼을 클릭합니다.
2. **1단계: 사업자등록번호 확인**
   - 사업자등록번호, 대표자명, 개업일자를 입력하고 **[진위 확인]** 버튼을 클릭합니다.
   - 국세청 API를 통해 정보의 유효성을 확인합니다.
3. **2단계: 계정 정보 입력**
   - 이메일 주소와 비밀번호를 입력합니다.
   - **[회원가입]** 버튼을 클릭하면 가입 신청이 완료됩니다.
   - 관리자의 승인 후 로그인이 가능합니다.

## 3. 로그인

1. 메인 페이지에서 **[로그인]** 버튼을 클릭합니다.
2. 로그인 유형을 선택합니다.
   - **기업 관리자:** 사업자등록번호와 비밀번호를 입력하여 로그인합니다.
   - **관리자:** 관리자 아이디와 비밀번호를 입력하여 로그인합니다.
3. **[로그인]** 버튼을 클릭하여 대시보드로 이동합니다.

## 4. 기업 관리자 페이지

### 4.1. 대시보드

- 로그인한 기업의 상세 정보를 확인할 수 있습니다.
- 회사명, 사업자등록번호, 대표자명, 연락처, 주소, 가입일, 계정 상태 등이 표시됩니다.

### 4.2. 기업 정보 관리

- 왼쪽 사이드바에서 **[기업 정보]** 메뉴를 클릭합니다.
- 회사명, 대표자명, 연락처, 주소 등 수정 가능한 정보를 변경할 수 있습니다.
- **[정보 저장]** 버튼을 클릭하여 변경 사항을 저장합니다.

## 5. 관리자 페이지

### 5.1. 대시보드

- 전체 등록 기업 수, 승인 완료 기업 수, 승인 대기 기업 수를 확인할 수 있습니다.
- 최근 승인된 기업 목록이 표시됩니다.

### 5.2. 기업 회원 승인

- 왼쪽 사이드바에서 **[기업 회원 승인]** 메뉴를 클릭합니다.
- 가입을 신청한 기업 목록을 확인할 수 있습니다.
- 각 기업 정보 우측의 **[승인]** 또는 **[거절]** 버튼을 클릭하여 처리합니다.

### 5.3. 기업 관리

- 왼쪽 사이드바에서 **[기업 관리]** 메뉴를 클릭합니다.
- 승인된 전체 기업 목록을 조회하고, 검색할 수 있습니다.
- 기업 계정의 상태를 **활성** 또는 **정지**로 변경할 수 있습니다.

### 5.4. 사용자 관리

- 왼쪽 사이드바에서 **[사용자 관리]** 메뉴를 클릭합니다.
- 등록된 관리자 계정 목록을 조회하고 관리할 수 있습니다.
